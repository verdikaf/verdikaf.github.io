    Internfo merupakan website pencarian dan pendaftaran magang yang dibuat untuk memenuhi tugas akhir semester. project yang dikerjakan secara berkelompok ini menggunakan framework Laravel sebagai backend, HTML, CSS dan Vue.js sebagai frontend, serta mySQL sebagai database.
